Detailed Description

Threadpool / worker pool.

This struct represents a threadpool instance: a collection of worker threads and an associated work queue.

Examples:
sum_example.c.

◆ threadpool_notify()

int threadpool_notify ( struct threadpool *  tp)

Notify any blocked processes that the threadpool state has changed.

Calling threadpool_notify() wakes up any processes that are blocked in calls to threadpool_wait_for_work_() or threadpool_wait_for_complete_(). These processes then re-evaluate the queue state, going back to sleep, or returning depending on the outcome.

Parameters
tpThe thread pool.

int threadpool_wait_for_work_ ( struct threadpool *  tp,
struct task *  t 
)
private

Block and wait for work to appear on the queue.

If there is a pending task on the queue, this function pops the task, and stores it at the location pointed to by t immediately. If no task is pending, this function blocks until there is a pending task and the queue is notified.

Parameters
tpThe thread pool.
tPointer at which to store the popped task.
Returns
0 on success, non-zero on failure.

◆ threadpool_worker_func_()

void * threadpool_worker_func_ ( void *  tpp)
private

Worker thread function for use with thread pool.

This function loops forever, consuming and executing tasks on the queue. When no work is available, this function will block on threadpool_wait_for_work_(). Call threadpool_notify() to wake up all blocked threads and resume task execution.

Parameters
tpThread pool to which worker thread belongs (casted to void *)
