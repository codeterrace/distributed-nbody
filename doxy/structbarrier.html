◆ barrier_wait()

int barrier_wait ( struct barrier *  b)

Synchronize calling thread with barrier.

The calling thread will block until the required number of threads have reached the barrier.

Parameters
bThe barrier.
Returns
BARRIER_SERIAL_THREAD if the calling thread did not block (calling thread was the final thread to reach the barrier), BARRIER_FINAL_THREAD if the calling thread was the final thread to exit from barrier_wait(), and zero otherwise.
