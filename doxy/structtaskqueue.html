Detailed Description

Task queue FIFO for assigning tasks to worker threads.

This struct represents a task queue instance, and all of its associated state.

◆ taskqueue_notify()

int taskqueue_notify ( struct taskqueue *  q)

Notify any blocked processes that the queue state has changed.

Calling taskqueue_notify() wakes up any processes that are blocked in calls to taskqueue_wait_for_work() or taskqueue_wait_for_complete(). These processes then re-evaluate the queue state, going back to sleep, or returning depending on the outcome.

Parameters
qThe task queue.

int taskqueue_wait_for_work ( struct taskqueue *  q,
struct task *  t 
)

Block and wait for work to appear on the queue.

If there is a pending task on the queue, this function pops the task, and stores it at the location pointed to by t immediately. If no task is pending, this function blocks until there is a pending task and the queue is notified.

Parameters
qThe task queue.
tPointer at which to store the popped task.
Returns
